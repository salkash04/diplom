Курс Java Basic

В этом курсе мы рассмотрим основы языка программирования Java, его структуру и основные концепции.

После завершения модуля ты узнаешь:

  • как работает язык Java и его основные принципы;
  • какие существуют типы данных и переменные в Java;
  • что такое операторы и как их использовать для управления логикой программы;
  • как создавать и использовать массивы и коллекции в Java;
  • что такое объектно-ориентированное программирование и как его применять на практике;
  • как создавать и использовать классы и объекты в Java;
  • как обрабатывать исключения и ошибки;

Состав модуля:

Модуль включает в себя теоретические лекции, тесты для закрепления информации, а также практические задания.
