Теория: Условные операторы

1. Условный оператор if-else

Условный оператор if-else позволяет выполнять различные блоки кода в зависимости от условия. Если условие истинно, выполняется блок кода внутри if; если ложно — выполняется блок кода внутри else.

if (условие) {
    // Код, который выполняется, если условие истинно
} else {
    // Код, который выполняется, если условие ложно
}

2. Условный оператор switch-case

Оператор switch-case используется для выбора одного из нескольких блоков кода для выполнения. Это удобно, когда у вас есть несколько значений для проверки.

switch (выражение) {
    case значение1:
        // Код для значения 1
        break;
    case значение2:
        // Код для значения 2
        break;
    default:
        // Код, который выполняется, если ни одно значение не совпадает
}

Важно помнить о ключевом слове break, которое завершает выполнение switch. Если его не использовать, выполнение продолжится до следующего case.
