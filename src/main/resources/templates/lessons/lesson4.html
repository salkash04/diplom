Урок 4: Работа с простыми типами данных

I. Работа с простыми типами данных:

    Примитивные типы:

    Java поддерживает несколько примитивных типов данных, каждый из которых имеет свой диапазон значений:

      byte: 8-битное целое число со знаком. Диапазон от -128 до 127. short: 16-битное целое число со знаком. Диапазон от -32,768 до 32,767. int: 32-битное целое число со знаком. Диапазон от -2,147,483,648 до 2,147,483,647. long: 64-битное целое число со знаком. Диапазон от -9,223,372,036,854,775,808 до 9,223,372,036,854,775,807. float: 32-битное число с плавающей запятой одинарной точности. Диапазон от приблизительно -3.4E+38 до 3.4E+38. double: 64-битное число с плавающей запятой двойной точности. Диапазон от приблизительно -1.7E+308 до 1.7E+308. char: 16-битный символ Unicode. Диапазон от '\u0000' до '\uffff'. boolean: Логический тип данных, который может принимать только два значения: true или false.
    Обертки для примитивных типов:

    Java предоставляет классы-обертки для каждого из примитивных типов:

      Integer, Float, Double, Boolean, Character, Byte, Short, и Long.

    Эти классы позволяют работать с примитивными типами как с объектами и поддерживают автоматическую упаковку (autoboxing) и распаковку (unboxing). Например:

    
// Автоупаковка
Integer a = 10; // int автоматически преобразуется в Integer

// Автораспаковка
int b = a; // Integer автоматически преобразуется в int

    Важно отметить, что классы-обертки обеспечивают доступ к методам для работы с соответствующими примитивными типами и могут использоваться в коллекциях.

    Строки (String):

    Строки в Java представляют собой последовательности символов и являются неизменяемыми. Вы можете создавать строки, конкатенировать их и манипулировать подстроками:

    
String str = "Hello";
str += " World"; // Конкатенация строк
System.out.println(str); // Вывод: Hello World

    Java также предоставляет классы StringBuilder и StringBuffer для работы с изменяемыми строками:

      StringBuilder: Используется для создания изменяемых строк и не синхронизирован (более быстрый). StringBuffer: Также используется для создания изменяемых строк, но синхронизирован (безопасен для многопоточной среды).
    Дата и время (java.time пакета):

    Пакет java.time предоставляет классы для работы с датами и временем:

      LocalDate: Представляет дату без времени. LocalTime: Представляет время без даты. LocalDateTime: Представляет дату и время. Instant: Представляет момент времени (timestamp). Duration: Представляет продолжительность между двумя временными точками. Period: Представляет период времени между двумя датами.

    Эти классы позволяют форматировать даты и время и вычислять интервалы между ними.
