Курс Spring Boot

Задача курса - подготовить к работе над коммерческим проектом в команде, дать представление об enterprise-разработке на Java, познакомить с технологиями: Tomcat, Maven, Hibernate, фреймворком Spring (а также Spring Boot) и его основными подфреймворками (MVC, Security, Data), шаблонизатором Thymeleaf, AJAX, REST, css-фреймворком Bootstrap. Вы научитесь настраивать конфигурации безопасности web-приложений, создавать адаптивные web-страницы для своего приложения, работать с современным http-шаблонизатором, создавать RESTful-сервисы, освоите современные фреймворки для быстрой разработки web-приложений, работающих с базами данных.

План курса "Spring Boot"

Модуль 1. Введение в Spring Boot и создание первого приложения

  • Что такое Spring Boot?
  • Основные преимущества использования Spring Boot.
  • Установка и настройка окружения для разработки.
  • Создание простого RESTful API.
  • Запуск приложения и работа с встроенным сервером (Tomcat).

Модуль 2. Конфигурация приложения и работа с базами данных

  • Работа с файлами конфигурации (application.properties и application.yml).
  • Введение в JPA и Hibernate.
  • Конфигурация подключения к базе данных.
  • Создание сущностей и репозиториев с использованием Spring Data JPA.

Модуль 3. Внедрение зависимостей и обработка ошибок

  • Основы внедрения зависимостей в Spring.
  • Использование аннотаций @Autowired, @Component, @Service, @Repository.
  • Глобальная обработка ошибок с использованием @ControllerAdvice.

Модуль 4. Работа с RESTful сервисами

  • Создание контроллеров с использованием аннотации @RestController.
  • Обработка HTTP-запросов (GET, POST, PUT, DELETE).
  • Управление кодами ответа HTTP.

Модуль 5. Безопасность и тестирование приложений

  • Введение в Spring Security: настройка аутентификации и авторизации.
  • Написание юнит-тестов с использованием JUnit и Mockito.
  • Интеграционное тестирование с использованием Spring Test.
